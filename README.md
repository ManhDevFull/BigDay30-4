## Cài Đặt
Hướng dẫn cách cài đặt dự án. Ví dụ:

```bash
git clone https://github.com/KhoaCongNgheSoUTE/tkw25.nhom16.git nhom16
cd nhom16
```

Trên Windows:
```bash
start index.html
```
Trên Linux:
```bash
xdg-open index.html
```
Trên MacOs
```bash
open index.html
```
